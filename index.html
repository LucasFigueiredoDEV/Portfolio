Sobre Mim

Eu sou um Desenvolvedor Full stack Brasileiro. Posso desenvolver sites responsivos do zero e elevá-los a experiências web modernas e amigáveis.

Transformar minha criatividade e conhecimento no desenvolvimento de sites e sistemas tem sido minha paixão por mais de um ano. Tenho auxiliado na manutenção e desenvolvimento de sistemas para hospitais por cerca de um ano. Sempre me esforço para aprender sobre as mais novas tecnologias e frameworks, pois aprender é a minha maior paixão.

Projeto - Processo Seletivo HCP

Descrição Detalhada:

Este projeto consiste em uma página web onde são realizadas todos os processos seletivos das unidades HCP Gestão.

Principais Tecnologias:

Laravel Framework
Bootstrap Framework
PHP
JavaScript

Neste Projeto auxiliei no desenvolvimento:

1 - Lobby de Realização de provas.

2 - Autenticação de Logins.

3 - APIs de comunicação via E-mail.

4 - Cálculo para geração de rankings.

5 - Validação de etapas de referente ao processo seletivo.

6 - Aba para cadastro de Processos Seletivos.

7 - Aba para cadastro de Pergunta, Respostas e Tempo de prova referente a cada Processo Seletivo.
